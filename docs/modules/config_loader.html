cl.load_config(script_config)
Loads and merges the appropriate conky configurations. Determines the current desktop environment and display server, loads the respective configuration tables, and merges them with the core configuration and the provided script configuration.

Parameters:

  • script_config table Custom configuration table to be merged on top of the defaults.

Returns:

    table The fully merged configuration table.
