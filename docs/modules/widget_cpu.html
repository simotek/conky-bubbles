Class Cpu

Polygon-style CPU usage & temperature indicator. Looks best for CPUs with 4 to 8 cores but also works for higher numbers.
Cpu:init(args)

Parameters:

  • args table of options
    • cores integer How many cores does your CPU have?
    • scale integer radius of central polygon
    • gap integer space between central polygon and outer segments
    • segment_size integer radial thickness of outer segments
    • rounded integer round edges for a circular look, default above 8 (boolean)

Class CpuRound

Round CPU usage & temperature indicator. Best suited for CPUs with high core counts.
CpuRound:init(args)

Parameters:

  • args table of options
    • cores integer How many cores does your CPU have?
    • inner_radius integer Size of inner circle
    • outer_radius integer Max radius for core at 100%
    • grid integer Number of grid lines to draw in the background. (optional)
